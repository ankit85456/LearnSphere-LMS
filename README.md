# 🎓 LearnSphere - Full-Stack E-Learning Platform

**LearnSphere** is a feature-rich, responsive Learning Management System (LMS) designed to bridge the gap between students and educators. It provides a seamless, animated, and interactive environment for course management, learning progression, and student engagement.

## 🚀 Key Features

* **Dual-Role Authentication:** Dedicated dashboards and functionalities for both **Students** and **Instructors**.
* **Interactive Dashboards:** 
  * *Students:* Track enrolled courses, view study hours, and check completion progress.
  * *Instructors:* Monitor active courses, total student enrollment, average ratings, and course earnings.
* **Modern UI/UX:** Styled using custom Vanilla CSS with fluid micro-animations (Framer Motion) and modern icons (Lucide React).
* **Robust Backend:** Secure user authentication using JSON Web Tokens (JWT) and bcrypt password hashing.
* **Database Integration:** Flexible and scalable data modeling using MongoDB and Mongoose.

## 🛠️ Tech Stack

* **Frontend:** React.js, React Router, Framer Motion, Lucide Icons
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT (JSON Web Tokens)
